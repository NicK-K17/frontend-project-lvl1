### Проект "Игры разума"
Проект представляет собой пять консольных игр, в которых необходимо дать правильный ответ в соответствии с задачей в конкретной игре. После трёх удачных ответов игра считается пройденной.

### Инструкция по установке
1. Установите глобально Node.js последней версии в систему.
2. Склонируйте репозиторий проекта в систему. 
3. Выполните установку зависимостей командой make install.
4. Запустите любую игру соответствующей командой из описания.

### Описание
1. Определение является ли предложенное число четным. Запуск игры командой brain-even.
2. Решение предложенной армифметической задачи. Запуск игры командой brain-calc.
3. Нахождение наибольшего общего делителя двух предложенных чисел. Запуск игры командой brain-gcd.
4. Поиск пропущенного числа в последовательности чисел. Запуск игры командой brain-progression.
5. Определение является ли предложенное число простыми. Запуск игры командой brain-prime.
